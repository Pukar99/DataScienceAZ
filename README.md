# DataScienceAZ

This repository contains activities related to Data Science in Python.

## Activities

1. Data Cleaning
2. Exploratory Data Analysis
3. Machine Learning Models
4. Data Visualization

Feel free to explore the code and contribute to the project!

## Getting Started

To get started with the activities, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/DataScienceAZ.git`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Run the Jupyter notebooks in the `notebooks` directory.
